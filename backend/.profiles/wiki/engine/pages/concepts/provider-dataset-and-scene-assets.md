# Provider, Dataset, and Scene Assets

- Provider APIs answer where a dataset comes from and how it is registered.
- View APIs answer how that dataset is shown once it has been found.
- Keeping provider and view questions separate prevents DFS and PlanetView answers from collapsing into one another.

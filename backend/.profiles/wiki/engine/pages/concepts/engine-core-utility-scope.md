# Engine Core Utility Scope

- Core utility APIs support filesystem, string, process, and generic helper tasks.
- In the current wiki routing, they should not absorb display, raster, coordinate, or scene-domain explanations.
- Use this page to keep general utility questions from leaking into the wrong API family.

## Answering Guidance
- Use this page to explain the shared idea or family boundary first.
- Route back to the owning workflow for exact API names, signatures, and code-level steps.
- Do not rely on this page alone when the answer needs executable call shapes.

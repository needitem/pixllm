# Spatial Reference and Coordinate Transform

- `XSpatialReference` answers what coordinate system a point or dataset uses.
- `XCoordinateTransformation` answers how a point moves from one spatial reference to another.
- Code-lookup APIs and SR-building APIs are prerequisites for stable transform answers.

## Answering Guidance
- Use this page to explain the shared idea or family boundary first.
- Route back to the owning workflow for exact API names, signatures, and code-level steps.
- Do not rely on this page alone when the answer needs executable call shapes.
